# Nathaniel Osborne Website

This repository now includes a root static website for GitHub Pages.

## Site entrypoint
- `index.html`
- `styles.css`
- `script.js`

## Features
- About section
- Contact section
- Live GitHub repositories list
- Search + language filters

## Deployment
The site deploys automatically through GitHub Actions using:
- `.github/workflows/pages.yml`

After pushing to `main`, enable Pages in GitHub repo settings if not already enabled:
- Settings -> Pages -> Source: GitHub Actions
